# ledstrip-control
Attiny85 controlled eased on-off led strip controller

## required components

    * 12V DC power supply
    * Attiny85
    * ULN2803 (darlington array)
    * 12V Ledstrip (2nd ledstrip optional)
    * 5V Voltage regulator
    * Capacitors (0.33uF and 0.1uF)
    * Resistor 1K (pull down)
    * Switch
    * PIR motion sensor
